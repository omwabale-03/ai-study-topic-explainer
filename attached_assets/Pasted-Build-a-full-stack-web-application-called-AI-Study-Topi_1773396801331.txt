Build a full-stack web application called **AI Study Topic Explainer** using the following **mandatory technology stack**:

TECH STACK REQUIREMENTS

* Next.js (App Router)
* TypeScript
* Tailwind CSS
* Gemini AI API (free tier)
* Deployable on Vercel

IMPORTANT:
Do NOT use Vite, Express, or any other framework.
The project must be a **Next.js App Router project**.

---

PROJECT DESCRIPTION

Create a web application where a student can enter a study topic and receive a **simple explanation generated by AI**.

Example:

Input:
Photosynthesis

Output:
Photosynthesis is the process by which plants make their own food using sunlight, carbon dioxide, and water.

---

FEATURES TO IMPLEMENT

1. Topic Input Page

Create a clean UI using Tailwind CSS.

The page should contain:

* Page title: **AI Study Topic Explainer**
* Text input field for entering a topic
* Button labeled **Explain Topic**

Example topics:

* Newton's Laws
* Photosynthesis
* Binary Search
* World War II

---

2. AI Explanation Generation

When the user clicks **Explain Topic**:

* Validate that the topic input is not empty
* Send a POST request to an API route
* The API route calls Gemini AI
* Gemini generates a **simple explanation for students**
* Return the explanation to the frontend
* Display the explanation on the page

Use prompt:

Explain the topic "<topic>" in simple terms for a school student in about 5 sentences.

---

3. Loading State

While waiting for the AI response, display:

Generating explanation...

---

4. Error Handling

Handle the following errors:

If topic input is empty:

Display message:
Please enter a topic to continue.

If AI request fails:

Display message:
Something went wrong. Please try again.

---

5. Explanation Display

Display the AI response clearly inside a styled card.

Example:

Topic: Photosynthesis

Explanation:
Photosynthesis is the process by which plants make their own food using sunlight, carbon dioxide, and water.

---

API ROUTE REQUIREMENT

Create an API route using Next.js App Router:

POST /api/explain

Input JSON:

{
"topic": "photosynthesis"
}

Output JSON:

{
"explanation": "Photosynthesis is the process by which plants..."
}

The API route must:

* Validate the topic
* Call Gemini AI
* Return explanation
* Handle errors properly
